# Instrucciones

---

1. **Añade un objeto JSON al final de esta lista.**
2. **Asegúrate de colocar una COMA ( , ) después del objeto anterior.**
3. **Reemplaza la información de las llaves** (`"nombre"`, `"linkedin"`, `"github"`, `"descripcion"`) **con tus propios datos.**

### Ejemplo

```json
{
  "nombre": "Tu Nombre Completo",
  "linkedin": "tu-usuario-de-linkedin",
  "github": "tu-usuario-de-github",
  "descripcion": "Una descripción corta sobre ti y/o un dato
  }
